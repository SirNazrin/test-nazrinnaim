Simple to-do list web application built with Laravel, Inertia.js, and Vue

## Key Features
- **Task CRUD**: Add, edit, and delete tasks.
- **Search**: Filter tasks in real time.

## How to setup

1. **Download or clone the project**
   - If you're using Laravel Herd, place the project inside your Herd directory.
   - If you're using Laragon or XAMPP, place the project inside the www or htdocs folder.

2. **Setup Backend**
   - `composer install`
   - `cp .env.example .env`
   - `php artisan key:generate`
   - `php artisan migrate`

3. **Setup Frontend**
   - `npm install`
   - `npm run dev`

4. **Run Server**
   - `php artisan serve`
